Welcome to Feater

Feater is a tool for rapid deployment of selected features of your web application to isolated testing or demo environments.

It’s tech-agnostic in terms of your project's language and dependencies.
This is possible because it uses Docker containers and Docker Compose configurations.

It’s open-source and is released under MIT license.
Therefore you can use it for free as well as modify it to suit your needs.

It’s easy to configure.
If your project has a Docker Compose setup and some build scripts you won't need much more to build it with Feater.

It’s easy to host.
It runs in Docker containers and you can use it on your local machine or set up a dedicated server.
